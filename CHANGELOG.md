# Changelog

All notable changes to `laravel-easy-dev` will be documented in this file.

## [2.0.0] - 2025-08-18

### Added
- 🎯 **Enhanced Interactive UI** - Beautiful command-line interface with progress bars and guided setup
- 🎮 **Interactive Mode** - Step-by-step wizard for CRUD generation
- 🏗️ **Service Layer Generation** - Automatic service layer with business logic separation
- 📝 **Smart Form Requests** - Intelligent validation rules based on field types and migration analysis
- 🔄 **Enhanced Relationship Detection** - Advanced migration parsing for better relationship discovery
- 🌐 **Improved API Support** - Better API controller generation with proper JSON responses
- 📚 **Beautiful Help Command** - Enhanced help system with examples and formatting
- 🎨 **Progress Tracking** - Visual progress bars for better user experience
- ⚙️ **Flexible Options** - More command options for customization

### Enhanced
- 🔍 **Migration Parser** - Improved regex patterns for modern Laravel syntax
- 🎮 **Controller Generation** - Better controller structure with dependency injection
- 🗄️ **Repository Pattern** - Enhanced repository with better interface contracts
- 🛣️ **Route Generation** - Smarter route handling with proper imports
- 📋 **Model Enhancement** - Better model enhancement with fillable detection

### Fixed
- 🐛 Controller import issues in route files
- 🐛 Fillable field duplication in models
- 🐛 Relationship detection for SQLite databases
- 🐛 Migration file parsing edge cases
